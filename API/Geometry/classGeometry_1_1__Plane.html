Detailed Description

template<typename T_>
class Geometry::_Plane< T_ >

A three dimensional plane. The plane is internally represented by its normal and offset from (0,0,0):

a*nx + b*ny + c*ny = offset

Note
Although the general definition allows a normal of arbitrary length – internal calculations request the normal to always be of length 1.0
